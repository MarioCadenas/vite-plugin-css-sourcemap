What to Check

  1. Build this playground: npm run build in the playground-scss-entrypoint directory
  2. Check the generated dist/assets/styles.css.map
  3. The sourcemap should include mappings for ALL SCSS partials:
    • _variables.scss
    • _reset.scss
    • _layout.scss
    • _buttons.scss
    • _cards.scss
    • _forms.scss
    • _utilities.scss
  4. Bug: Currently, sourcemaps only cover main.scss and not the imported partials
